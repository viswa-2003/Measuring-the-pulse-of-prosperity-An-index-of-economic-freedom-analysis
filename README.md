📊 Measuring the Pulse of Prosperity
An Index of Economic Freedom Analysis



📌 Project Overview
This project analyzes the Index of Economic Freedom to understand how economic policies influence a country’s prosperity, growth, and global competitiveness.

Using data visualization and analytical techniques, the dashboard provides insights into:

Economic Freedom Scores

Regional Comparisons

GDP vs Freedom Correlation

Rule of Law & Government Size Analysis

Business & Trade Freedom Trends

The goal is to measure how economic freedom impacts national development.

🎯 Objectives
Analyze global economic freedom data

Identify patterns and trends across regions

Compare developed vs developing nations

Understand correlation between economic freedom and GDP

Create an interactive visual dashboard for insights
